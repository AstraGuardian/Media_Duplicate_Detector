# Code of Conduct

## Our Commitment

This project is a welcoming space for everyone. Whether you're a seasoned developer or just learning, whether you're contributing code or asking questions‚Äîyou're welcome here.

## Expected Behavior

- **Be respectful** - Treat others how you'd want to be treated
- **Be constructive** - Offer helpful feedback, not criticism
- **Be patient** - Everyone has different skill levels and time constraints
- **Be inclusive** - Welcome diverse perspectives and backgrounds
- **Be understanding** - We all make mistakes; focus on learning

## Unacceptable Behavior

- Harassment, insults, or discriminatory comments
- Personal attacks or trolling
- Publishing others' private information
- Spamming or self-promotion
- Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies to:
- Issues and pull requests
- Code comments and documentation
- Any project-related communication

## Enforcement

If someone violates this Code of Conduct:
1. They'll be asked to stop
2. Repeated violations may result in being blocked from the project
3. Severe violations (harassment, threats) will result in immediate removal

## Reporting

If you experience or witness unacceptable behavior:
- For minor issues: Address it politely in the thread
- For serious issues: Open a private issue or contact the maintainer directly

## Attribution

This Code of Conduct is adapted from common open source community standards, tailored for a small, friendly project.

We're all here because we care about making this project better. Let's keep it friendly, constructive, and welcoming.
